# GoGaijin – Tokyo Services for Foreigners

**GoGaijin** is a mobile app that helps foreigners (gaijin) in Tokyo easily find essential services such as healthcare, dining, housing, banking, and more — with English-speaking staff.

---

## 📞 Support

If you need help using GoGaijin or have any questions:

* Open a [support issue on GitHub](https://github.com/lopezwebdev/GoGaijin-Support/issues)
* Or contact us by email: [info@gogaijin.me](mailto:info@gogaijin.me)
